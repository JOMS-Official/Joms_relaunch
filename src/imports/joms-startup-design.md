Design a premium futuristic startup website for JOMS (Just One More Step).

Style inspiration: Apple minimalism + Stripe clarity + Linear futuristic UI.

The website must feel like a high-potential technology startup attracting investors, builders, and early believers.

Design principles:
	•	Minimal yet premium
	•	Glassmorphism UI
	•	Futuristic visuals
	•	Clear storytelling
	•	Strong investor credibility
	•	Excellent mobile experience

The site should immediately communicate that JOMS builds next-generation digital products including mobile platforms and marketplaces.

	•	border 1px rgba(255,255,255,0.15)
	•	soft shadows
